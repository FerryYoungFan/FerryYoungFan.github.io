Beamforming Introduction

Parameters:

NAnt (N): number of antennae

dAnt (d): distance between two adjacent antennae

c0 (c): light speed ≈ 3x108 m/s

λ0 (λ): radio wavelength

ω0: radio wave angular velocity: ω0 = 2πc0 / λ0

ΔT: unit delay (in second), negative value stands for unit lead

ω0 ΔT: beam pattern steering phase

θ: angle refers to antenna plane normal vector (from -180° to +180°)
